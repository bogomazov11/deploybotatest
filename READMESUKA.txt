Всё, что тебе нужно на сервере - питон и определенные библиотеки для запуска бота
Устанавливается всё через терминал(убедись, что на твоем сервере есть Python 3.10 или выше, роли особо не играет)

pip install python-telegram-bot==13.7
pip install requirements.txt -r

Запуск:

в директории бота открой терминал.

python tg.py

если нет никаких логов(после запуска просто пустой терминал с курсором внизу) - всё работает.
Ты мог наблюдать на моих скринах логи с message id. Фактически это и станет сигналом, что бот работает и отправляет сообщения другим юзерам.

На этом по сути всё.
